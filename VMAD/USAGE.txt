================================================================================
VMAD — Faithful Video Reproduction via Progressive Velocity Distillation
Version 2.0.0
================================================================================

目标:
  给定参考视频 V_ref，找到最优条件表示 (Δe, η_inv)，使预训练 T2V 模型
  生成的视频 V_gen 最大化与 V_ref 的多维度相似性 (CLIP/XCLIP/SSIM/LPIPS)。

三层渐进编码:
  Layer 1 (Text τ):       ~10² bits → 粗粒度语义引导 (跨模型迁移)
  Layer 2 (Embedding Δe): ~10⁴ bits → 中频动态重建 (核心创新, 待实验)
  Layer 3 (Noise η_inv):  ~10⁶ bits → 全结构保真 (同模型同种子)

当前进展:
  ✓ Layer 1 (Prompt 优化): V4 自动化 pipeline 已完成
    - 位置权重实验: 发现 DiT position 0 有 10-15× 注意力权重
    - V4 iter1 达到: CLIP 0.8842 (+1.39%), XCLIP 0.7430 (+2.66%)
  ✓ Layer 3 (噪声先验): SVD noise prior 已验证
    - α=0.001 即可提升 CLIP +0.7% (0.8912)
  ○ Layer 2 (嵌入优化): 代码已实现, 实验待运行
    - velocity_matching.py: Position-Aware Velocity Field Matching
    - 预期突破 Layer 1 天花板 (连续空间信息容量 >> 离散文本)

================================================================================
1. EXTRACT — 从参考视频提取重建资产
================================================================================

基本用法:
  python run_extract.py --video <input_video> --output <output_dir>

模式选择:

  # 默认: 忠实复现模式 (T_m=1.0, λ_dis=0.0)
  python run_extract.py --video input.mp4 --output ./output/repro

  # 可选: 运动迁移模式 (T_m=0.3, λ_dis=0.1)
  python run_extract.py --video input.mp4 --output ./output/motion \
      --T_m 0.3 --lambda_dis 0.1

模块开关 (默认全部启用, 用 --no-xxx 关闭):
  --no-inversion        关闭 Flow Matching Inversion
  --no-svd              关闭 SVD 频谱分析
  --no-blend            关闭噪声先验注入
  --no-velocity         关闭 Velocity Field Matching (Layer 2)
  --no-disentangle      关闭内容解耦损失
  --no-text_decode      关闭 VLM 文本解码
  --midpoint            使用 Midpoint 求解器 (默认 Euler)

超参数:
  --T_m FLOAT           优化时间范围 (默认: 1.0, 运动迁移用 0.3)
  --num_opt_steps N     优化步数 (默认: 100)
  --lr FLOAT            学习率 (默认: 1e-3)
  --lambda_dis FLOAT    解耦权重 (默认: 0.0, 运动迁移用 0.1)
  --alpha FLOAT         噪声混合比例 (默认: 0.001)
  --rho_s FLOAT         SVD 空间阈值 (默认: 0.0, 运动迁移用 0.1)
  --rho_m FLOAT         SVD 时间阈值 (默认: 1.0)

================================================================================
2. APPLY — 应用资产生成视频
================================================================================

基本用法:
  python run_apply.py --asset <asset_dir> --content "<prompt>" --output <output_dir>

  # 忠实复现 (使用原始 caption)
  python run_apply.py --asset ./output/repro/asset \
      --content "original caption here" --output ./output/reproduced

  # 运动迁移 (使用新内容)
  python run_apply.py --asset ./output/motion/asset \
      --content "a white cat" --output ./output/cat_motion
