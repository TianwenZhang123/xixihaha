# L2 噪声先验改进方向：论文调研 & 原创算法设计

> 创建时间: 2025-06-11  
> 背景: P-Flow L2 层（SVD noise prior）的 one-shot linear blend 已触及天花板（α=0.004, CLIP 0.8964, XCLIP 0.7874）。方向 C（频域重塑）、D（SGA 自适应 alpha）、E（PODI 正交分解）均告失败。本文档调研满足**严格黑盒约束**的噪声先验改进方案，并提出原创算法候选。

---

## 一、约束条件（黑盒边界）

P-Flow 架构中 T2V 模型（Wan2.1）处于**完全冻结**状态，因此任何 L2 改进方案必须严格满足：

1. **不能对 T2V 做额外 forward pass**（无 inversion、无 DDIM backward、无 score estimation）
2. **不能获取 T2V 内部梯度**（无 backprop、无 attention map、无 guidance loss）
3. **不能访问 T2V 的中间层/注意力权重**（无 cross-attention injection）
4. **只能操控 T2V 的输入端**：初始噪声 z_T 和 text prompt（以及 guidance scale 等超参）
5. **必须保持 T2V 单次推理流程不变**（不改步数、不改 scheduler、不加 callback）

合法操作空间：在噪声 z_T 送入 T2V 之前，对其进行任意预处理（blend、reshape、filter、decompose 等）。

### 2.3 失败根因

1. **对齐度系统性极低** (alignment ≈ 0.19~0.35)：prompt embedding 映射到 16 维 channel 空间后，与 η_temporal 的 channel 分布几乎正交。理论随机对齐度 = 1/√16 ≈ 0.25，实测值未超过这个水平，说明 text embedding → latent channel 之间**不存在有意义的语义映射关系**。

2. **parallel 分量能量极低** (parallel_std ≈ 0.06~0.11)：η_temporal 被分解后，"对齐"分量只保留了原始信号的 7%~11% 的标准差。renorm 到 std=1 后虽然幅度恢复，但方向信息已被严重扭曲——本质上是在注入一个几乎随机方向的噪声。

3. **cos(mixed, random) ≈ 1.0000**：最终混合噪声与纯随机噪声的余弦相似度达到 1.0000（四位小数），说明 PODI 的注入信号**在最终噪声中完全不可见**——α=0.004 本身就极弱（direction_shift ≈ 0.025），PODI 再削去 80%~90% 后剩余信号几乎为零。

4. **核心矛盾**：PODI 假设"prompt 的 channel-space 投影方向"携带有意义的语义信息。但 text encoder 的隐层维度 (4096) 到 latent channel 维度 (16) 的映射是通过简单的 chunked pooling 实现的，无法保证任何语义结构。

### 2.4 教训总结

| 路线 | 核心假设 | 验证结果 | 教训 |
|------|------|------|------|
| 方向 C (频域) | 频谱形状 = 运动节奏 | 频谱近平坦，信号太弱 | 间接信息不够 |
| 方向 D (SGA) | std = 质量代理 | 差异太小 (0.28~0.41) | 微调量无效 |
| 方向 E (PODI) | prompt→channel 有语义映射 | alignment ≈ 随机水平 | 映射不存在 |

**共同教训**: 在 α=0.004 linear blend 框架下，任何对 η_temporal "内容"的进一步筛选/过滤都会让本已微弱的信号更加无效。方向应转向：(a) 找到更强/更直接的时序信号源，或 (b) 改变注入方式使信号效力放大。

## 三、论文调研（满足黑盒约束）

### 3.1 Φ-Noise: Noise-Aware Temporal Priors for Video Generation (2025)

- **来源**: 2025 preprint
- **核心思想**: 基于 flow-matching 理论推导，证明满足特定 temporal correlation 结构的噪声能显著提升时序一致性。提出 Φ-structured noise，其 temporal correlation matrix 由视频的 optical flow 场决定。
- **满足黑盒？**: ✅ 只修改 z_T 的生成过程，不碰模型
- **关键公式**: z_T ~ N(0, Σ_Φ)，其中 Σ_Φ 的 frame-frame 相关性由 flow magnitude 决定
- **与我们的关系**: 
  - 相比我们的 SVD linear blend，Φ-Noise 提供了一个理论 grounded 的 temporal correlation 注入方式
  - 但原文依赖 optical flow 估计（额外计算），且针对 flow-matching 模型推导
  - **可借鉴**: 用 η_temporal 的 frame-frame 协方差矩阵构造相关噪声，而非直接 blend 内容

### 3.2 How I Warped Your Noise: a Temporally-Correlated Noise Prior for Diffusion Models (ICLR 2024 Oral)

- **来源**: ICLR 2024 Oral
- **核心思想**: 对标准高斯噪声施加 temporal warping（光流 warp），使相邻帧噪声沿运动方向相关，同时保持每帧边缘分布为 N(0,1)。
- **满足黑盒？**: ✅ 只修改初始噪声的生成方式
- **关键方法**: 
  - 生成一帧基础噪声 z_0
  - 用参考视频的光流 F_{t→t+1} 对 z_0 做 warp 得到 z_t
  - 在 non-overlapping 区域填充新采样噪声
  - 整体保持 marginal Gaussian
- **与我们的关系**:
  - 我们的 SVD η_temporal 已经隐式包含了 temporal correlation，但注入方式（linear blend）不保证结构性
  - WarpedNoise 提供了更优雅的注入范式：不是"把信号混进去"，而是"让噪声自身具有时序结构"
  - **可借鉴**: 用参考视频的帧间对应关系（不需要精确光流，SVD 低秩近似即可）构造 warped noise

### 3.3 FreqPrior: Frequency-Adapted Noise Prior for Video Diffusion (ICLR 2025)

- **来源**: ICLR 2025
- **核心思想**: 分析发现自然视频的 3D 频谱呈 power-law 衰减，但标准高斯噪声频谱是平坦的。提出在频域对噪声做 power-law 整形，使其频谱先验与真实视频分布对齐。
- **满足黑盒？**: ✅ 只在 z_T 生成阶段做频域变换
- **关键公式**: |Z(f)|² ∝ f^{-β}，β ≈ 2 for natural videos
- **与我们的关系**:
  - 我们的方向 C（频域重塑）失败的原因是"只改时间维度频谱形状"，信号太弱
  - FreqPrior 是**3D 频域**整形（时间+空间），力度更大
  - **区别**: FreqPrior 用的是通用视频频谱先验（无需参考视频），而我们有参考视频的特异性信息
  - **可借鉴**: 在 3D 频域做整形而非仅 temporal 维度；结合参考视频的特异性频谱而非通用 power-law

### 3.4 PYoCo: Preserve Your Own Correlation (ICCV 2023)

- **来源**: ICCV 2023
- **核心思想**: 为视频扩散模型设计 correlated noise prior，使 noise frames 之间具有受控的相关性。提出 mixed noise (shared + per-frame) 和 progressive noise (AR 生成)。
- **满足黑盒？**: ✅ 只改噪声初始化策略
- **关键方法**:
  - Mixed noise: z_t = √ρ·z_shared + √(1-ρ)·z_t_indep
  - Progressive noise: z_t = √ρ·z_{t-1} + √(1-ρ)·ε_t (AR)
- **与我们的关系**:
  - PYoCo 的 "shared noise" 思路可视为我们 SVD 低秩近似的极端简化版
  - 它证明了"帧间噪声相关性"对视频生成有帮助
  - **可借鉴**: 用 η_temporal 的 SVD 第一主成分作为 shared component（类似 PYoCo 的 z_shared），而非全量 blend

### 3.5 FreeInit: Bridging Initialization Gap in Video Diffusion (ECCV 2024)

- **来源**: ECCV 2024
- **核心思想**: 发现视频扩散模型的初始噪声中，低频分量决定了全局语义/运动，高频分量决定细节。提出迭代式 refinement：先生成 → 取低频 → 混合新高频噪声 → 重新去噪。
- **满足黑盒？**: ❌ **需要多次 forward pass**（iterative refinement）
- **与我们的关系**:
  - FreeInit 的"低频保留"洞察印证了我们的理解：temporal prior 的价值在低频
  - 但其迭代方案违反单次推理约束
  - **可借鉴**: 低频/高频分离思想——我们可以只在噪声的低频部分注入 temporal prior（一次性完成，无需迭代）

### 3.6 Video-MSG: Multi-Scale Guided Noise Prior (2025)

- **来源**: 2025 preprint
- **核心思想**: 对初始噪声进行 multi-scale 分解（Gaussian pyramid），在不同尺度注入不同强度的结构信息。低尺度（粗粒度）注入全局运动方向，高尺度（细粒度）保持随机性。
- **满足黑盒？**: ✅ 只预处理 z_T
- **关键洞察**: 不同空间尺度对模型的影响力不同——低分辨率层控制全局结构/运动，高分辨率层控制局部纹理
- **与我们的关系**:
  - 直接解决了我们"α=0.004 信号太弱"的问题——不是对全分辨率 blend α=0.004，而是在低尺度 blend 更大的 α（因为低尺度控制运动，高尺度保持随机性保证质量）
  - **核心借鉴**: 多尺度策略使得有效注入量可以在特定尺度上大幅提高，而不破坏整体噪声质量

### 3.7 Factorized Diffusion: Perceptual Illusions by Noise Decomposition (ECCV 2024)

- **来源**: ECCV 2024
- **核心思想**: 将噪声分解为多个正交子空间，不同子空间可以携带不同的条件信号，模型在去噪过程中自动从不同子空间提取对应信息。
- **满足黑盒？**: ✅ 只操控 z_T 的构造方式
- **与我们的关系**:
  - 提供了一个理论框架：噪声的不同"维度/子空间"对生成结果有差异化影响
  - PODI 的失败在于选错了子空间（channel-space prompt 投影）
  - **可借鉴**: 寻找正确的"有意义子空间"——应该基于 SVD 的时序特征本身（如前几个主成分方向）而非外部 prompt 映射

## 四、核心洞察与设计原则

基于前序 5 轮失败实验 + 论文调研，提炼出以下设计原则：

### 4.1 "注入什么"比"注入多少"重要，但信号必须足够强

- SGA 证明: 微调 α (±27%) 无法突破噪声底
- PODI 证明: 过度过滤让信号趋零（parallel_std < 0.1 → 注入不可见）
- 新方案的信号必须保持 **direction_shift > 0.05**（baseline 的 2x 以上）才能在模型内部被"感知"

### 4.2 多尺度 / 分频策略是关键杠杆

- 方向 C 证明: 仅 temporal 频域整形太弱
- Video-MSG / FreqPrior / FreeInit 共同指向: **在空间低频注入强信号，空间高频保持随机**
- 原理: 扩散模型的低频→高频生成顺序意味着低频噪声结构有更大的"杠杆效应"

### 4.3 帧间相关性注入优于内容注入

- PYoCo / WarpedNoise / Φ-Noise 证明: "让噪声帧间相关"比"把信号混入每帧"更安全高效
- 原理: 帧间相关性不改变每帧边缘分布（保持 N(0,1)），只影响联合分布
- 这解释了 baseline α=0.004 的成功: 它隐式引入了微弱的帧间相关性（通过 η_temporal 的 low-rank structure）

### 4.4 保持 marginal Gaussian 是安全性的充要条件

- WarpedNoise 严格证明了: 只要每帧边缘分布是 N(0,1)，模型不会崩溃
- 方向 C 的 catastrophic failure (S7 XCLIP -69%) 正是因为频域整形破坏了 Gaussian property
- **新方案必须 provably 保持 N(0,1) marginals**

## 五、原创算法方案

### 方案 F: TPI — Temporal Phase Injection (时间相位注入)

#### 核心思想

不改噪声的幅度谱（保证 Gaussian marginals），只在 3D 频域的**时间维相位**中注入参考视频的运动信息。

#### 算法

```
输入: η_temporal (SVD 滤波后), η_random (标准高斯)
参数: γ ∈ [0, 1] (相位注入强度)

1. F_ref = rFFT3D(η_temporal)              # 参考信号的 3D 频谱
2. F_rand = rFFT3D(η_random)               # 随机噪声的 3D 频谱
3. 分离: A_rand = |F_rand|, φ_rand = angle(F_rand)
4. 提取 temporal phase pattern:
   - φ_ref = angle(F_ref)
   - 只取时间频率 > 0 的分量 (排除 DC)
5. Phase blending:
   - φ_out = (1-γ)·φ_rand + γ·φ_ref       # 相位插值
6. 重建: F_out = A_rand × exp(i·φ_out)     # 保留随机噪声的幅度
7. η_out = IRFFT3D(F_out)
8. η_out = (η_out - mean) / std            # renorm to N(0,1)

输出: η_out — 时间维相位携带参考运动信息，幅度保持随机
```

#### 为什么有效

- **与方向 C 的区别**: 方向 C 改的是时间维**幅度**（spectrum shape），本方案改的是**相位**。信号处理理论表明，相位比幅度携带更多结构信息（经典实验: 交换两张图的幅度和相位，视觉内容跟随相位走）。
- **信号强度**: 相位改变不受"频谱近平坦"的限制——即使频谱平坦，相位仍可以携带丰富的时序结构。
- **安全性**: 保留 |F_rand| 意味着功率谱不变，renorm 后保证 N(0,1)。

#### 风险

- 3D FFT 的相位插值可能引入 artifact（相位 wrapping 问题）
- γ 过大时 renorm 可能需要大幅缩放，等效于破坏信号

#### 论文叙事

"Temporal Phase Injection" — 基于信号处理中"相位决定结构"的经典理论，我们在频域保留噪声的幅度谱（保证 Gaussian property），同时将参考视频的时间相位结构注入，使初始噪声在保持统计安全的同时获得时序运动引导。

### 方案 G: MSTDI — Multi-Scale Temporal Decomposition Injection (多尺度时序分解注入)

#### 核心思想

将噪声空间分为多个尺度（Gaussian pyramid），在不同空间尺度注入不同强度的 temporal prior——**低尺度注入强信号（控制全局运动）**，**高尺度保持随机（保证视觉质量）**。

#### 算法

```
输入: η_temporal (SVD 滤波后), η_random (标准高斯)
参数: 
  - num_levels = 4 (金字塔层数)
  - α_schedule = [0.05, 0.02, 0.004, 0.0] (各层 alpha, 从粗到细递减)

1. 构建 η_temporal 的 Gaussian Pyramid:
   - L0_ref = downsample(η_temporal, 8x)  # 最粗层: [F, C, H/8, W/8]
   - L1_ref = downsample(η_temporal, 4x)  # [F, C, H/4, W/4]
   - L2_ref = downsample(η_temporal, 2x)  # [F, C, H/2, W/2]
   - L3_ref = η_temporal                  # 原始分辨率

2. 构建 η_random 的 Gaussian Pyramid (同上)

3. 逐层 blend (从粗到细):
   - L0_out = √α[0]·L0_ref + √(1-α[0])·L0_rand   # α=0.05, 强注入
   - L1_out = √α[1]·L1_ref + √(1-α[1])·L1_rand   # α=0.02
   - L2_out = √α[2]·L2_ref + √(1-α[2])·L2_rand   # α=0.004
   - L3_out = L3_rand                              # α=0.0, 纯随机

4. Laplacian 重建:
   - η_out = L3_out + (L2_out - upsample(L1_out)) + (L1_out - upsample(L0_out)) + L0_out
   - (标准 Laplacian pyramid 重建)

5. η_out = (η_out - mean) / std            # renorm to N(0,1)

输出: η_out — 低频部分强时序引导，高频部分保持随机
```

#### 为什么有效

- **解决"α=0.004 信号太弱"**: 在最粗层 α=0.05（是 baseline 的 12.5x），但因为只影响低频结构，不会破坏视觉质量。
- **杠杆效应**: 扩散模型先处理低频再处理高频。低频噪声中的结构会被模型"放大"为全局运动模式，而高频保持随机确保细节不退化。
- **安全性论证**: 
  - S7/S31 在 α=0.01 崩塌的原因是全分辨率注入过强 → MSTDI 只在低频注入强信号，高频保持安全
  - 类比: 在 1/8 分辨率(30×40)上 α=0.05 注入 ≈ 只改变了总像素的 1.5% 的噪声结构，整体 direction_shift 仍可控

#### 风险

- Gaussian pyramid 的 downsample/upsample 可能引入插值 artifact
- Laplacian 重建后需要严格 renorm，可能损失一些结构
- α_schedule 需要搜索

#### 论文叙事

"Multi-Scale Temporal Prior Injection" — 受 FreeInit 的低频主导洞察和 Video-MSG 的多尺度策略启发，我们设计了金字塔式注入：在粗粒度空间尺度强注入 temporal prior 以引导全局运动，在细粒度保持随机以保证视觉质量。这利用了扩散模型"coarse-to-fine"生成特性的杠杆效应。

### 方案 H: CEGI — Channel-Energy Gated Injection (通道能量门控注入)

#### 核心思想

η_temporal 的 16 个 channel 并非同等重要。某些 channel 集中了时序运动信息（高 temporal variance），而另一些 channel 主要是空间纹理（低 temporal variance）。CEGI 按 channel 的时序能量选择性注入：**只在时序能量高的 channel 注入，其余保持随机**。

#### 算法

```
输入: η_temporal [F, C, H, W], η_random [F, C, H, W]
参数: 
  - top_k: 要注入的 channel 数 (默认 4, 即 top-25%)
  - α_inject: 被选中 channel 的注入强度 (默认 0.02, 是 baseline 的 5x)

1. 计算每个 channel 的时序能量:
   - temporal_var[c] = Var(η_temporal[:, c, :, :], dim=0).mean()
   # 即: 在 frame 维度上的方差，再空间平均

2. 排序选择 top-k channels:
   - selected = argsort(temporal_var, descending=True)[:top_k]

3. 选择性注入:
   - For c in selected:
       η_out[:, c, :, :] = √α_inject · η_temporal[:, c, :, :] + √(1-α_inject) · η_random[:, c, :, :]
   - For c NOT in selected:
       η_out[:, c, :, :] = η_random[:, c, :, :]  # 保持纯随机

4. Per-channel renorm:
   - For each c: η_out[:, c, :, :] = (η_out[:, c, :, :] - mean) / std

输出: η_out — 时序信息集中的 channel 被强注入，其余 channel 保持随机
```

#### 为什么有效

- **信号集中**: 不再把 α=0.004 稀释到全部 16 个 channel，而是把 α=0.02 集中在最有效的 4 个 channel。有效信号强度提升 5x × (16/4) = 20x。
- **与 PODI 的区别**: 
  - PODI 用 prompt embedding 选择方向 → 映射不存在，alignment ≈ 随机
  - CEGI 用数据本身（temporal variance）选择 channel → 完全数据驱动，不依赖任何外部映射
- **安全性**: 未被选中的 12 个 channel 完全是纯随机噪声，为模型提供了充分的"安全缓冲"。per-channel renorm 保证每个 channel 都是 N(0,1)。
- **direction_shift 估算**: 4/16 channel 上 α=0.02 → 整体 direction_shift ≈ √0.02 × 4/16 ≈ 0.035，远大于 baseline 的 0.025。

#### 风险

- Latent space 的不同 channel 是否有明确的"时序/空间"分工尚不确定（需要实证）
- 如果所有 channel 的 temporal_var 都相近，选择就没有意义
- top_k 和 α_inject 需要搜索

#### 论文叙事

"Channel-Selective Temporal Prior Injection" — 我们发现 latent space 中不同 channel 的时序响应有显著差异。通过识别 temporal-dominant channels 并集中注入 prior，我们在保持整体噪声安全性的同时大幅提升了有效注入强度。

### 方案 I: OCS — Orthogonal Complement Suppression (正交补空间抑制)

#### 核心思想

反转 PODI 的逻辑：不是"只注入对齐分量"（信号太弱），而是"**在随机噪声中抑制与 temporal prior 正交的分量**"。即保留 η_random 中与 η_temporal 空间结构相似的部分，抑制完全无关的部分。

#### 算法

```
输入: η_temporal [F, C, H, W], η_random [F, C, H, W]
参数: 
  - k: SVD 保留的主成分数 (默认 3)
  - suppress_ratio ∈ [0, 1]: 正交空间的抑制比例 (默认 0.5)

1. 对 η_temporal 做 SVD 提取主子空间:
   - 将 η_temporal reshape 为 [F, C*H*W]
   - U, S, V = SVD(η_temporal_reshaped)
   - V_k = V[:k, :]  # 前 k 个右奇异向量 = η_temporal 的主方向

2. 将 η_random 在 V_k 子空间上分解:
   - η_random_flat = η_random.reshape(F, -1)
   - proj = η_random_flat @ V_k.T @ V_k       # 在主子空间的投影
   - orth = η_random_flat - proj               # 正交补空间

3. 抑制正交补:
   - η_out_flat = proj + (1 - suppress_ratio) × orth
   
4. Reshape + renorm:
   - η_out = η_out_flat.reshape(F, C, H, W)
   - η_out = (η_out - mean) / std             # renorm to N(0,1)

输出: η_out — 保留了与 η_temporal 主方向一致的随机分量，抑制了无关分量
```

#### 为什么有效

- **与 PODI 的互补关系**: 
  - PODI: 从 η_temporal 中提取信号 → 信号太弱
  - OCS: 从 η_random 中抑制噪声 → 噪声被减弱后，信号相对增强
  - OCS 不注入 η_temporal 的内容，而是让 η_random "更倾向于" temporal 主方向
- **信号强度**: suppress_ratio=0.5 时，正交方向的能量减半。对于 k=3 的子空间，大约保留了 3/(C×H×W) 的方向能量不变、其余方向减半。这是一个温和的调整，但通过 renorm 后有效放大了主方向的比重。
- **安全性**: renorm 保证 N(0,1)。由于只抑制（不注入），不会引入 η_temporal 的"有毒内容"。

#### 风险

- η_temporal 的 SVD 前 k 个主成分方向可能不稳定（对不同样本差异大）
- suppress_ratio 过大会让噪声严重偏离 Gaussian（renorm 前分布形状改变）
- 计算成本略高（需要对 [F, C×H×W] 做 SVD）

#### 论文叙事

"Subspace-Guided Noise Shaping" — 我们从 SVD temporal prior 中提取主成分子空间，然后在随机噪声中抑制与该子空间正交的分量。这种"减法"策略避免了直接注入 temporal content 的毒性风险，同时通过 subspace alignment 为扩散模型的去噪过程提供了隐式的运动方向引导。

## 六、方案对比与优先级推荐

| 方案 | 信号机制 | 预期 direction_shift | 安全性 | 实现复杂度 | 优先级 |
|------|------|:---:|:---:|:---:|:---:|
| F: TPI | 时间相位注入 | 中 (0.03~0.06) | 高 (幅度不变) | 中 | ⭐⭐⭐ |
| G: MSTDI | 多尺度分层注入 | 高 (0.05~0.10) | 高 (高频保持随机) | 中 | ⭐⭐⭐⭐ |
| H: CEGI | 通道能量选择性注入 | 高 (0.03~0.05) | 中 (依赖 channel 分工) | 低 | ⭐⭐⭐⭐⭐ |
| I: OCS | 正交补抑制 | 中 (0.03~0.04) | 高 (不注入内容) | 低 | ⭐⭐⭐ |

### 推荐实验顺序

1. **H: CEGI** (最优先)
   - 实现最简单（只是 per-channel 的条件分支）
   - 理论清晰（temporal variance = 运动 channel）
   - 信号强度大幅提升（集中 α 到少数 channel）
   - 可快速验证"选择性注入"思路是否成立

2. **G: MSTDI** (第二优先)
   - 理论支撑最强（多篇论文验证了多尺度策略）
   - 解决了"α 太弱"的根本问题（低频 α=0.05）
   - 需要 Gaussian pyramid 实现，略复杂

3. **F: TPI** (第三)
   - 理论新颖（相位注入在视频噪声先验中尚未被探索）
   - 但 3D FFT 相位操作实现复杂，debug 周期长

4. **I: OCS** (第四)
   - 概念有趣但效果不确定——"抑制无关方向"的收益可能不如"注入有效方向"
