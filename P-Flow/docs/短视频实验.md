## 2026-06-28: SVD+FI 三层联合门控最终版 (ABC) ✅ 已完成

### 最终配置

```
SVD: α=0.004, ρ_s=0.1, ρ_m=0.9
  ├── η_std 双向门控: Floor(η_std<0.32→α=0.006), CAP(η_std>0.45→α=0.002)
  └── 方向A SVD联动跳过: mean_cos>0.08 → α=0 (FI跳过时关闭SVD)

FI: λ_max=0.10, layers=mid, schedule=middle_peak
  ├── Quality Gate: mean_cos>0.08 → FI跳过
  ├── Adaptive Gate: temp=5.0, 上限 0.30 (方向C)
  └── 累计预算: max_norm=10000, decay_min=0.3 (方向B)
```

### 门控逻辑架构

```
原始噪声 η_random
    │
┌───▼──────────────────────────────────────┐
│ SVD 模块                                  │
│   1. η_std 双向门控 (Floor/CAP)            │
│   2. 方向A: mean_cos>0.08 → α=0           │
│   输出: η = √α·η_temporal + √(1-α)·η_random│
└──────────────────────────────────────────┘
    │
┌───▼──────────────────────────────────────┐
│ FI 模块 (U-Net mid layers 10-19)          │
│   Layer1: Quality Gate (mean_cos阈值)      │
│   Layer2: Schedule (middle_peak时间曲线)    │
│   Layer3: Adaptive Gate (逐层cos_sim上限)   │
│   Layer4: 预算控制 (累计norm衰减)           │
│   输出: h = (1-λ)·h + λ·h_ref              │
└──────────────────────────────────────────┘
```
