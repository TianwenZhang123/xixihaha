# P-Flow 逐层验证复现指南

> 从零开始，逐层叠加 Layer 1 → Layer 3，验证每一层的独立贡献。
>
> 环境：A800 GPU，Wan2.1-T2V-1.3B，480×832，81 frames，15fps。

## 1. Step 0：Baseline（纯 VLM Caption 直出）

Baseline 不使用任何改动点 flag，直接用 VLM 的原始 caption 生成视频。

```bash
python run.py \
    --data_dir /root/autodl-tmp/data/video-200/water_mark_out \
    --caption_dir /root/autodl-tmp/data/video-200/captions_qwen \
    --output_dir /root/autodl-tmp/outputs/step0_baseline \
    --sample_ids $SAMPLE_IDS \
    --seed 42 \
    --resume
```

**预期结果**：CLIP ≈ 0.8703，XCLIP ≈ 0.7164

---

## 2. Step 1：+Layer 1（V4 Hybrid Prompt Rewrite）

Layer 1 是外部预处理步骤——用 LLM 将 VLM 描述性 caption 改写为指导性 prompt。

### 2.1 生成 Hybrid Caption

```bash
python scripts/rewrite_hybrid.py \
    --input-dir /root/autodl-tmp/data/video-200/captions_qwen \
    --output-dir /root/autodl-tmp/data/video-200/captions_hybrid \
    --backend dashscope \
    --model qwen-plus \
    --sample-ids $SAMPLE_IDS \
    --skip-existing
```

改写三原则：
1. 首词 = 主体名词（利用 UMT5 position 0 权重优势）
2. 注入时序动作链（initially...then...finally...）
3. 保留原始视觉描述词（颜色、材质、光线等不替换）

### 2.2 用 Hybrid Caption 生成视频

```bash
python run.py \
    --data_dir /root/autodl-tmp/data/video-200/water_mark_out \
    --caption_dir /root/autodl-tmp/data/video-200/captions_hybrid \
    --output_dir /root/autodl-tmp/outputs/step1_L1_hybrid \
    --sample_ids $SAMPLE_IDS \
    --seed 42 \
    --resume
```

**预期结果**：CLIP ≈ 0.8842，XCLIP ≈ 0.7430（相对 Baseline：CLIP +1.6%，XCLIP +3.7%）

---

## 3. Step 2：+Layer 1 + Layer 2（SVD Noise Prior）

Layer 2 在 Layer 1 的基础上叠加噪声先验：从参考视频反演噪声 → SVD 分离运动成分 → 与随机噪声混合。

```bash
python run.py \
    --data_dir /root/autodl-tmp/data/video-200/water_mark_out \
    --caption_dir /root/autodl-tmp/data/video-200/captions_hybrid \
    --output_dir /root/autodl-tmp/outputs/step2_L1L2_noise_prior \
    --noise_prior \
    --alpha 0.004 \
    --sample_ids $SAMPLE_IDS \
    --seed 42 \
    --resume
```

`--noise_prior` 是快捷 flag，等价于 `--inversion --svd --blend`。

关键参数：
- `--alpha 0.004`：最优混合系数（5 点网格搜索确定，√α≈0.063 的运动信号权重）
- `--inversion_steps 50`：反演 ODE 步数（默认值，标准精度）
- `--rho_s 0.1`：空间 SVD 截断阈值（去内容）
- `--rho_m 0.9`：时间 SVD 保留阈值（保运动）

**预期结果**：CLIP ≈ 0.8953，XCLIP ≈ 0.7667（相对 Baseline：CLIP +2.9%，XCLIP +7.0%）

---

## 4. Step 3：+Layer 1 + Layer 2 + Layer 3（Velocity Field Matching）

Layer 3 在前两层基础上叠加速度场匹配：优化一个 embedding 残差 Δe，让生成过程中的速度场对齐参考视频定义的理想轨迹。

### 4.1 基础 Velocity（v1 配置）

```bash
python run.py \
    --data_dir /root/autodl-tmp/data/video-200/water_mark_out \
    --caption_dir /root/autodl-tmp/data/video-200/captions_hybrid \
    --output_dir /root/autodl-tmp/outputs/step3_L1L2L3v1 \
    --velocity_full \
    --alpha 0.004 \
    --embed_strength 0.02 \
    --sample_ids $SAMPLE_IDS \
    --seed 42 \
    --resume
```

`--velocity_full` 等价于 `--inversion --svd --blend --velocity`。

关键参数：
- `--embed_strength 0.02`：Δe 注入强度（理论推导 + 网格验证确定的最优值）
- `--velocity_steps 30`：优化步数（条件反演起点质量高，30 步足够收敛）
- `--velocity_lr 1e-3`：Adam 学习率 + Cosine Annealing

**预期结果**：CLIP ≈ 0.8981，XCLIP ≈ 0.7705（相对 Baseline：CLIP +3.2%，XCLIP +7.6%）

### 4.2 增强 Velocity v2（当前最优配置）

在 v1 基础上启用分层采样、Padding Mask 和运动加权：

```bash
python run.py \
    --data_dir /root/autodl-tmp/data/video-200/water_mark_out \
    --caption_dir /root/autodl-tmp/data/video-200/captions_hybrid \
    --output_dir /root/autodl-tmp/outputs/step3_L1L2L3v2_best \
    --velocity_full \
    --alpha 0.004 \
    --embed_strength 0.02 \
    --velocity_K 4 \
    --velocity_motion_weight 1.0 \
    --sample_ids $SAMPLE_IDS \
    --seed 42 \
    --resume
```

v2 相对 v1 的改进：
- `--velocity_K 4`：分层多时间步采样，覆盖密度 4×，梯度方差下降 ~2×
- `--velocity_motion_weight 1.0`：运动感知 Loss 加权，高运动区域获更高权重
- Padding-Aware 梯度 Mask（自动生效，无需额外 flag）

**预期结果**：CLIP ≈ 0.8998，XCLIP ≈ 0.7736（相对 Baseline：CLIP +3.4%，XCLIP +8.0%）

## 5. 评估

每一步完成后，使用统一评估脚本计算 CLIP 和 X-CLIP 指标：

```bash
python evaluation/run_clip_xclip_eval.py \
    --orig-dir /root/autodl-tmp/data/video-200/water_mark_out \
    --gen-dir /root/autodl-tmp/outputs/<STEP_OUTPUT_DIR> \
    --caption-dir /root/autodl-tmp/data/video-200/captions_hybrid \
    --output-dir /root/autodl-tmp/outputs/<STEP_OUTPUT_DIR>/eval_clip
```

将 `<STEP_OUTPUT_DIR>` 替换为对应步骤的输出目录名。

### 5.1 一键评估所有步骤

```bash
#!/bin/bash
ORIG_DIR="/root/autodl-tmp/data/video-200/water_mark_out"
CAPTION_DIR="/root/autodl-tmp/data/video-200/captions_hybrid"

for step_dir in step0_baseline step1_L1_hybrid step2_L1L2_noise_prior step3_L1L2L3v1 step3_L1L2L3v2_best; do
    echo "====== 评估: $step_dir ======"
    python evaluation/run_clip_xclip_eval.py \
        --orig-dir $ORIG_DIR \
        --gen-dir /root/autodl-tmp/outputs/$step_dir \
        --caption-dir $CAPTION_DIR \
        --output-dir /root/autodl-tmp/outputs/$step_dir/eval_clip
    echo ""
done
```

## 7. 常见问题

### Q1：alpha 的默认值和最优值不一致？

`run.py` 中 `--alpha` 默认值为 0.001（历史保守值），但实验验证最优值为 0.004。运行时需显式指定 `--alpha 0.004`。

### Q2：embed_strength 默认值和最优值不一致？

同理，`run.py` 中 `--embed_strength` 默认值为 0.005（初始设定），最优值为 0.02。需显式指定。

### Q3：Velocity 是否依赖 Noise Prior？

`--velocity` 需要 `--inversion`（因为 Δe 优化依赖反演噪声 η_inv 来构建目标速度场 v* = z₀ - η_inv）。如果只传 `--velocity` 而不传 `--inversion`，程序会自动启用 inversion 并打印警告。

建议直接用 `--velocity_full`（= --inversion --svd --blend --velocity）。

### Q4：如何只跑单个视频做快速测试？

```bash
python run.py \
    --video /root/autodl-tmp/data/video-200/water_mark_out/31.mp4 \
    --caption "$(cat /root/autodl-tmp/data/video-200/captions_hybrid/31.txt)" \
    --velocity_full \
    --alpha 0.004 \
    --embed_strength 0.02 \
    --output_dir /root/autodl-tmp/outputs/single_test
```

### Q5：如何恢复中断的批量实验？

加 `--resume` flag，程序会跳过已有输出的样本。

### Q6：每一步的计算开销？

| 配置 | DiT Equivalent Forwards | 相对 Baseline |
|------|------------------------|--------------|
| Baseline | ~30 | 1.0× |
| +Noise Prior | ~80 | 2.7× |
| +Velocity | ~170 | 5.7× |

## 8. 一键复现脚本

将以上所有步骤整合为一个完整的 shell 脚本：

```bash
#!/bin/bash
# P-Flow 逐层验证一键复现脚本
# 使用方法: bash docs/reproduce.sh

set -e

# ── 配置 ──
DATA_DIR="/root/autodl-tmp/data/video-200/water_mark_out"
CAPTION_QWEN="/root/autodl-tmp/data/video-200/captions_qwen"
CAPTION_HYBRID="/root/autodl-tmp/data/video-200/captions_hybrid"
OUTPUT_BASE="/root/autodl-tmp/outputs"
SAMPLE_IDS="7 17 21 31 32 33 34 43 46 47"
SEED=42

echo "============================================"
echo " P-Flow 逐层验证复现"
echo " 样本: $SAMPLE_IDS"
echo " 种子: $SEED"
echo "============================================"
echo ""

# ── Step 0: Baseline ──
echo "[Step 0] Baseline (VLM caption 直出)..."
python run.py \
    --data_dir $DATA_DIR \
    --caption_dir $CAPTION_QWEN \
    --output_dir $OUTPUT_BASE/step0_baseline \
    --sample_ids $SAMPLE_IDS \
    --seed $SEED --resume

# ── Step 1: Layer 1 (Hybrid Prompt Rewrite) ──
echo ""
echo "[Step 1] Layer 1: Hybrid Prompt Rewrite..."

# 1a. 生成 hybrid caption (如果不存在)
python scripts/rewrite_hybrid.py \
    --input-dir $CAPTION_QWEN \
    --output-dir $CAPTION_HYBRID \
    --backend dashscope --model qwen-plus \
    --sample-ids $SAMPLE_IDS \
    --skip-existing

# 1b. 用 hybrid caption 生成视频
python run.py \
    --data_dir $DATA_DIR \
    --caption_dir $CAPTION_HYBRID \
    --output_dir $OUTPUT_BASE/step1_L1_hybrid \
    --sample_ids $SAMPLE_IDS \
    --seed $SEED --resume

# ── Step 2: Layer 1 + Layer 2 (Noise Prior) ──
echo ""
echo "[Step 2] Layer 1 + Layer 2: SVD Noise Prior (alpha=0.004)..."
python run.py \
    --data_dir $DATA_DIR \
    --caption_dir $CAPTION_HYBRID \
    --output_dir $OUTPUT_BASE/step2_L1L2_noise_prior \
    --noise_prior --alpha 0.004 \
    --sample_ids $SAMPLE_IDS \
    --seed $SEED --resume

# ── Step 3a: L1 + L2 + L3v1 (Velocity) ──
echo ""
echo "[Step 3a] Layer 1 + Layer 2 + Layer 3v1: Velocity Matching..."
python run.py \
    --data_dir $DATA_DIR \
    --caption_dir $CAPTION_HYBRID \
    --output_dir $OUTPUT_BASE/step3_L1L2L3v1 \
    --velocity_full --alpha 0.004 --embed_strength 0.02 \
    --sample_ids $SAMPLE_IDS \
    --seed $SEED --resume

# ── Step 3b: L1 + L2 + L3v2 (Best Config) ──
echo ""
echo "[Step 3b] Layer 1 + Layer 2 + Layer 3v2: Best Config..."
python run.py \
    --data_dir $DATA_DIR \
    --caption_dir $CAPTION_HYBRID \
    --output_dir $OUTPUT_BASE/step3_L1L2L3v2_best \
    --velocity_full --alpha 0.004 --embed_strength 0.02 \
    --velocity_K 4 --velocity_motion_weight 1.0 \
    --sample_ids $SAMPLE_IDS \
    --seed $SEED --resume

# ── 统一评估 ──
echo ""
echo "============================================"
echo " 开始评估..."
echo "============================================"

for step_dir in step0_baseline step1_L1_hybrid step2_L1L2_noise_prior step3_L1L2L3v1 step3_L1L2L3v2_best; do
    echo ""
    echo ">>> 评估: $step_dir"
    python evaluation/run_clip_xclip_eval.py \
        --orig-dir $DATA_DIR \
        --gen-dir $OUTPUT_BASE/$step_dir \
        --caption-dir $CAPTION_HYBRID \
        --output-dir $OUTPUT_BASE/$step_dir/eval_clip
done

echo ""
echo "============================================"
echo " 全部完成！结果保存在 $OUTPUT_BASE 下各 eval_clip/ 目录"
echo "============================================"
```
