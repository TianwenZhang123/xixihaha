================================================================================
                        P-Flow 使用说明
================================================================================

用法:
    python run.py [输入参数] [改动点开关] [参数调节] [模型路径] [执行控制]


================================================================================
  输入参数
================================================================================

  --video VIDEO           单个参考视频路径
  --caption CAPTION       初始 caption (为空则由 VLM 自动生成)
  --data_dir DATA_DIR     批量模式: 视频目录 (文件名为 0.mp4, 1.mp4, ...)
  --caption_dir CAPTION_DIR
                          批量模式: caption 目录 (文件名为 0.txt, 1.txt, ...)
  --output_dir OUTPUT_DIR 输出目录 (默认: outputs)
  --sample_ids ID [ID ...]
                          指定处理哪些样本 ID
  --limit N               最多处理 N 个样本


================================================================================
  改动点开关 (核心 — 消融实验用这些 flag 排列组合)
================================================================================

  --inversion             启用 Flow Matching Inversion
                          从参考视频反演噪声，获取运动轨迹信息

  --svd                   启用 SVD 两阶段滤波
                          空间维度去内容 (ρ_s) + 时间维度保运动 (ρ_m)

  --blend                 启用噪声混合
                          η = √α · η_temporal + √(1-α) · η_random

  --iter N                启用迭代 VLM 优化，N 为迭代轮数 (0=不迭代)
                          每轮: 生成视频 → VLM 对比分析 → 优化 prompt

  --midpoint              使用二阶中点法 ODE 求解器 (替代默认 Euler)
                          更高精度的反演，减少截断误差

  --composite             启用三面板垂直拼接
                          将 ref|prev|current 拼成一张送 VLM 对比


================================================================================
  快捷组合
================================================================================

  --noise_prior           等价于 --inversion --svd --blend
  --full                  等价于 --inversion --svd --blend --iter 10 --composite


================================================================================
  参数调节
================================================================================

  --alpha ALPHA           噪声混合权重 (默认: 0.001)
  --rho_s RHO_S           空间 SVD 阈值 — 越小去内容越彻底 (默认: 0.1)
  --rho_m RHO_M           时间 SVD 阈值 — 越大保运动越多 (默认: 0.9)
  --steps STEPS           推理步数 (默认: 30)
  --guidance GUIDANCE     CFG scale (默认: 5.0)
  --seed SEED             随机种子 (默认: 42)
  --height HEIGHT         视频高度 (默认: 480)
  --width WIDTH           视频宽度 (默认: 832)
  --num_frames NUM_FRAMES 帧数 (默认: 81)
  --fps FPS               帧率 (默认: 15)


================================================================================
  模型路径
================================================================================

  --model_path PATH       Wan2.1-T2V-1.3B 模型路径
                          (默认: /root/autodl-tmp/models/Wan2.1-T2V-1.3B-Diffusers)
  --vlm_path PATH         VLM 模型路径
                          (默认: /root/models/Qwen2.5-VL-7B-Instruct)
  --vlm_provider {local,dashscope,mock}
                          VLM 后端 (默认: local)


================================================================================
  执行控制
================================================================================

  --resume                跳过已有输出 (断点续跑)
  --verbose               打印详细日志


================================================================================
  使用示例
================================================================================

  # 1. 纯 baseline (无任何改动，caption → 一次生成)
  python run.py --video ref.mp4 --caption "a cat walking on grass"

  # 2. 只开噪声先验 (inversion + svd + blend)
  python run.py --video ref.mp4 --caption "a cat" --inversion --svd --blend

  # 3. 噪声先验 + 迭代优化 10 轮
  python run.py --video ref.mp4 --caption "a cat" --inversion --svd --blend --iter 10

  # 4. 全部改动点 (快捷方式)
  python run.py --video ref.mp4 --caption "a cat" --full

  # 5. 用中点法替代 Euler
  python run.py --video ref.mp4 --caption "a cat" --noise_prior --midpoint

  # 6. 批量处理 + 断点续跑
  python run.py --data_dir ./videos --caption_dir ./captions --full --resume

  # 7. 消融: 只开 inversion (不做 SVD 和 blend)
  python run.py --video ref.mp4 --caption "a cat" --inversion

  # 8. 消融: 只开迭代 (不用噪声先验)
  python run.py --video ref.mp4 --caption "a cat" --iter 10

  # 9. 调参: 修改 alpha 和 SVD 阈值
  python run.py --video ref.mp4 --caption "a cat" --full --alpha 0.01 --rho_s 0.2 --rho_m 0.8
